The datasets are the following:

*Ground truth* : 

 - CavBench/datasets/ground_truth_csv.zip
 - CavBench/datasets/ground_truth_xml.zip
 
*KVFinder* : 

 - CavBench/datasets/kvfinder_csv.zip
 - CavBench/datasets/kvfinder_xml.zip
